Detailed Description

FB19 Subscriber configuration structure.

Field Documentation

◆ busAddress

int16_t busAddress

Bus address of this Subscriber instance. Its range is FB19_BUS_ADDR_SUBS_LOWEST to FB19_BUS_ADDR_SUBS_HIGHEST, which is currently 1 to 4 for the no cost version. Note that it can change over the lifetime of the Bus Subcriber instance, since it is the Bus Controller that assignes the bus addresses.
