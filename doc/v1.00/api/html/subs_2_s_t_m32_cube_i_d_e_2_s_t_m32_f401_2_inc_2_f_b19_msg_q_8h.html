int FB19MsgQ_init ( FB19MsgQ_t *  queue)

This function is used to initialize a message queue. All messages in the queue will be be initialized with 0, including ALL their internal data structures. It is no error, if the capacity of the queue is 0; in that case no operation will be performed on the queue. If the capacity is a negative number however, an error will be returned.

Parameters
[in]msgpoints to the message queue that shall be initialized.
Returns
R_SUCCESS if the parameter checks pass, R_ERROR otherwise.
