This function is used to initialize a message queue. All messages in the queue will be be initialized with {\ttfamily 0}, including ALL their internal data structures. It is no error, if the capacity of the queue is 0; in that case no operation will be performed on the queue. If the capacity is a negative number however, an error will be returned.
