# Bus Controllers and Bus Subscribers Examples

On a FieldBus19 system, there are 1 Bus Controller and 1 to 30 Bus Subscribers attached.

Select the level of details you want to see by looking at the numbers in the file names. **The higher the number** at the end of the name, **the more details** and examples the files contain.

Please note that these files just contain code snippets, not compilable code.

## Bus Controllers
The files `FB19Ctrl_bareMetal_<n>.c` show how to turn an embedded application into a Bus Controller.

## Bus Subscribers
The files `FB19Subs_bareMetal_<n>.c` show how to turn an embedded application into a Bus Subscriber.

Thanks in advance for reporting bugs and typos to the author of the files.

Have fun!
