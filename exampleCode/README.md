# Bus Controllers and Bus Subscribers Examples

On a FieldBus19 system, there are 1 Bus Controller and 1 to 30 Bus Subscribers attached.

The files `FB19Ctrl_bareMetal_<n>.c` show how to turn an embedded application into a Bus Controller. The higher the number at the end of the name, the more details and examples the file contains.

The files `FB19Subs_bareMetal_<n>.c` show how to turn an embedded application into a Bus Subscriber. The higher the number at the end of the name, the more details and examples the file contains.

Thanks in advance for reporting bugs and typos to the author of the files.
